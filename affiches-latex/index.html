Affiches scientifiques en LaTeX (en)

Pour réaliser mon poster Paedophile Activity in a Large P2P System , j'ai utilisé le template beamerposter, voici un peu d'aide pour utiliser ce template.

Beamerposter

Description

Le poster se présente en mode portrait, avec un en-tête contenant titre et logos des affiliations, puis un espace pour le contenu organisé avec deux colonnes contenant des blocs, avec un titre sur fond plus sombre et un contenu sur fond légèrement gris. Par défaut, les titres sont bleus, avec possibilité d'avoir des blocs rouges ou verts (avec \alertblock et \exempleblock, classiques en beamer). La page se termine par une ligne de pied-de-page (footer) contenant les URL et les adresses mail des auteurs, surmonté d'une ligne de remerciements pour les bailleurs de fond (ANR et Europe dans mon cas).

Téléchargement et documentation

  • Mon archive se trouve ici : lien. Normalement, après l'avoir téléchargée et en avoir extrait les fichiers, vous devriez être capable d'obtenir le même pdf que moi via la commande :

    $ pdflatex poster.tex

  • Le template original se récupère à cette adresse : http://www-i6.informatik.rwth-aachen.de/~dreuw/latexbeamerposter.php. L'un des avantages de ce poster est qu'il y a un groupe Google avec une partie questions/réponses où les développeurs ont déjà répondu à un certain nombre de questions (par exemple, comment insérer des flêches entre des blocs et surtout comment outrepasser certains bugs) : http://groups.google.com/group/beamerposter

Avertissements

  • Ce template se compile avec pdflatex, il faut donc avoir les figures et logos dans un format jpeg, png ou pdf (on peut faire les choses rapidement dans un terminal avec le programme 'epstopdf' :

    $ for file in $(ls *.eps); do epstopdf $file; done

  • Il y a deux lignes de texte en rouge au-dessus des colonnes (ça a son importance pour la taille des colonnes, voir dans le source).

  • Les flèches entre les blocs, ce n'est pas très facile à faire ! Il y a un certain nombre de problèmes de positionnement relatif des objets. Le sujet est en partie traité sur le googlegroup (et dans les commentaires du code). Dans le code, les flèches sont traitées dans des fichiers distincts, portant le nom sections/arrows_<nom-du-bloc-d'origine>.tex

Usage

Voici la structure commentée de mon dossier :

. 
├── beamerposter.sty # le template principal de beamerposter 
├── beamerthemeRF.sty # le template de theme, qui gère les couleurs et la mise en page (titres, header, footer, etc.)
├── figures # les courbes, au format pdf (et eps) 
├── logos # les logos utiles 
├── Makefile # peu rempli, je m'en sers pour un 'make clean' qui retire les fichiers inutiles après compilation (d'après leur extension, voir dans le fichier pour la liste de celles-ci). 
├── Old # plutôt que de faire des 'rm' indésirables, un 'mv $file Old' peut être utile  
├── poster.bib # le bibtex 
├── poster.tex # le fichier source principal, qui décrit le squelette du document 
├── sections # les fichiers sources de ce qui est dans les blocs 
└── textpos.sty # pour positionner les flèches (en surcouche du contenu)
